Markdown语法

发表于 2020-02-19 更新于 2020-02-24

[TOC]

安装插件

  1. 安装插件Markdown All in One，包含了最常用的Markdown优化。
  2. 安装插件Markdown Preview Github Styling，专门针对github pages的预览，功能有限，但是正是我需要的
  3. 如果不是针对github的预览，则可以安装Markdown Preview Enhanced，这个插件应用更普遍

Markdown语法

  1. 标题

    标题有共有六个等级，在前面加上一到六个 “#”

  2. 正文

    正文中想要换行，必须要多跳一行，如果在代码中只换一行，那么其实没有换行

  3. 代码

    正文中的代码块，在前后加上”`”，如果是一段代码段落，前后分别加上”```”

    注：可以根据不同的语言配置不同的代码着色

  4. 列表

    有序列表：输入数字 + 一个点”.” + 一个空格

    无序列表：输入”-“/“*”/“+” + 一个空格

    • 无序列表
      • 与前面的表示符号无关
        • 只与缩紧行数相关

  5. 加粗、倾斜和删除

    加粗

    倾斜

    删除线

  6. 引用

    此处是引用

    此处是多层引用第一层

    此处是多层引用第二层

    多层嵌套第一层

    多层嵌套第二层

    多层嵌套第三层

  7. 插入连接

    Markdown中文文档

  8. 插入图片

    • 可以在图片上传到github后，用github上图片的地址链接，这样网页上可以正常显示
    • 可以把包含图片的文件夹docs放到本地/hexo/public/images里，用相对路径/images/docs/images.jpg来调用

  9. 文字的个性设置
    可以直接用html语法对正文进行编辑，达到想要的展示效果

    • 居中：

      这一行居中

    • 换色 + 变化大小：

      接下来就是见证奇迹的时刻
      我可以设置这一句的颜色哈哈
      我还可以设置这一句的大小嘻嘻
      我甚至可以设置这一句的颜色和大小呵呵

  10. 数学公式
    主题目录下/hexo/theme/next/_config.yml设置mathjax里的enable: true
    并且在需要使用数学公式的博客里打开公式开关：
