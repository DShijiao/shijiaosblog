linux常用命令

发表于 2020-02-19 更新于 2022-04-01 分类于 京东
  1. 快速查看端口占用情况lsof -i tcp:8080或者netstat -nlp
  2. 查看PID具体信息ll /proc/PID
  3. 同样，查看CPU和内存信息可以使用cat /proc/cupinfo 和 cat /proc/meminfo。更详细CPU信息查询可以使用cat /proc/cpuinfo | grep name | cut -f2 -d: | uniq -c 和cat /proc/cpuinfo | grep physical | uniq -c
  4. 查看显卡信息可以使用命令lspci | grep -i vga
  5. 查看磁盘使用情况df -k或df -m，分别以kb或mb为单位，查看磁盘使用量
  6. GPU当前使用状况：nvidia-smi
  7. 每10秒，更新一次GPU情况：watch -n 10 nvidia-smi
  8. free -h看内存情况

参考文章：Linux系统查看CPU、机器型号、内存等信息

使用vscode原创连接服务器

发表于 2020-02-19 更新于 2022-03-24 分类于 京东

使用vscode原创连接服务器

安装

  1. 在左侧“扩展”里选择并安装扩展 Remote-SSH
  2. 点击左侧新增的扩展按钮，并点击左上角“+”号，然后会出现config的配置，按如下配置
    Host name-as-you-like
    HostName 11.12.205.19
    User root
  3. 此时配置好后尝试登陆，会需要输入密码
  4. 可以用配置成不需要密码
    1. 本地建个ssh ssh-keygen -t rsa -b 4096 -f ~/.ssh/id_rsa-remote-ssh，会生成id_rsa-remote-ssh和id_rsa-remote-ssh.pub两个文件
    2. 将id_rsa-remote-ssh.pub配到服务器的authorized_keys（一般在～/.ssh/）文件里
    3. 将id_rsa-remote-ssh配到本地的config中，即修改刚刚的config成如下
      Host name-as-you-like
      HostName 11.12.205.19
      User root
      IdentityFile ~/.ssh/id_rsa-remote-ssh
  5. 可以连接远程服务器了

Bert

Layer Normalization

Bert中使用的是Layer Normalization，源于对batch normalization的改进。
batch normalization的优点：

  • 加快神经网络训练时间
  • 对学习率有更好的容忍
  • 重新拉正数据分布，避免进入激活函数的饱和区域，缓解梯度消失
    batch normalization的缺点：
  • 对batch size的大小比较敏感
  • 在RNN和LSTM等序列模型中效果不太好
    针对以上两个BN的缺点，提出了LN，较好的解决了上面两个问题
